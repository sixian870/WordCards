# 單字卡程序

提供使用者一個兼具單字管理、即時發音、自動播放與隨機複習的客製化單字卡。


## 核心功能

### 1. 單字複習系統

* **自動播放模式 (Auto Play)**： 整合內建 Timer 機制與 Windows Media Player 元件，實現單字的自動切換與發音播放。
  
* **隨機洗牌機制 (Shuffle)**： 一鍵即可將所有單字順序完全打亂，有效打破大腦對順序的死記硬背，提升長期記憶效率。

* **一鍵字母排序 (Sort A-Z)**： 隨時能將混亂的單字清單重新依據英文字母 A-Z 進行重新排序，方便使用者查找單字。

### 2. 單字庫管理

* **文字檔串流技術**： 動態讀寫 WordCards.txt 檔案，完美落實資料持久化（Persistence），確保每一次新增、修改的單字資料與音檔路徑都能妥善保存。

### 3. 使用者體驗

* **無滑鼠操作**： 全程支援使用者透過 `Tab` 跳轉 `Space` 確定。
  
* **清晰介面**： 透過不同顏色與字體大小分區方便使用者查看。
  
* **畫面縮放**： 透過元件設定確保使用者如何縮放畫面都不會變形。


## 使用說明


1.  **主畫面功能**：

    | 功能 | 操作方式 | 效果 |
    | :------ | :------ | :---- |
    | **手動瀏覽** | 點擊左側清單中的單字 | 畫面即時顯示該單字的「拼字」、「音標」與「中文解釋」，並同步播放該單字讀音。 |
    | **自動播放** | 點擊 `Play` 按鈕 | 程式會自動跳轉至下一個單字顯示畫面並發音。按鈕字樣變更為 `Stop`，再點擊一次即可暫停。 |
    | **打亂順序** | 點擊 `打亂` 按鈕 | 清單順序立即打亂，若原本處於自動播放狀態，系統會貼心地自動暫停播放，避免畫面錯亂。 |
    | **按字母排序** | 點擊 `排序` 按鈕 | 無論清單目前多混亂，皆會一鍵變成標準的字母 A-Z 排列。 |
    | **鍵盤快捷鍵** | 按下 `Enter` / 按下 `Space` | `Enter`： 切換至下一個單字並發音。<br>Space： 重複播放當前選取單字的音檔。<br>* 須確保焦點不停在 `打亂` 、 `排序` 按鈕上 |

2.  **單字編輯畫面**：

* 雙擊主畫面左側清單中要編輯的單字，會開啟「編輯單字視窗」，焦點自動聚焦音標。
    
* 除單字固定不能修改外，其餘音標、音檔路徑、解釋可以修改。

* 修改完後點擊 `儲存` 按鈕，更新將立即寫入記憶體與單字檔案中。
